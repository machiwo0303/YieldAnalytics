# 📘 YieldAnalytics — 日本株 高配当 × 財務 × 成長 スクリーニングシステム

YieldAnalytics は、日本株の **高配当銘柄を “配当・財務・成長・規模・CF・減配リスク” の6軸で総合評価**し、  
**買うべき銘柄だけを自動抽出する分析ツール**です。

本プロジェクトは以下の 3 ステップで構成されています：

1. **データ取得**（jpx_screening_fetch.py）  
2. **スコアリング**（jpx_scoring_engine.py）  
3. **最終フィルタリング**（jpx_yield_filter.py）

---

# 🏗️ 1. jpx_screening_fetch.py — データ取得

## 概要
TOPIX・JPX400・任意の銘柄リストから：

- 10年分の配当履歴  
- 財務諸表（PL/BS/CF）  
- ROE / EPS / 時価総額  
- 特別配当補正済みの配当データ  
- 安全な yfinance アクセス（info の None 対策・リトライ・スキップ）

を取得し、**output_financial_dividend_merged.csv** を生成します。

---

## 主な特徴

### ✔ 特別配当補正（normalize_special_dividends）
記念配当・特別配当による “見かけ上の減配” を自動補正。

```
prev < cur(特別) > next の場合、cur を前後の最大値に補正
```

→ 翌年の配当が通常に戻っても **減配扱いにならない**。

---

### ✔ 安全な info 取得（safe_get_info）
yfinance の不安定なレスポンス（None / 壊れた JSON）を完全に回避。

- 例外処理  
- リトライ  
- dict 以外は破棄  
- 空 dict の場合は銘柄スキップ  

---

### ✔ 最新年の減配誤判定を防止
最新年は配当が確定していないため、  
**最新年の減配は判定対象から除外**。

---

### 出力ファイル
```
output_financial_dividend_merged.csv
```

---

# 🧮 2. jpx_scoring_engine.py — スコアリング

## 概要
取得したデータを基に、銘柄ごとに **最大120点の総合スコア** を算出します。

---

## スコア構成

| 項目 | 点数 | 内容 |
|------|------|------|
| 配当スコア | 最大40 | 10年配当、増配傾向、減配リスク |
| 財務スコア | 最大30 | 自己資本比率、営業CF、現金比率 |
| 成長スコア | 最大30 | 売上推移、EPS、ROE |
| 規模スコア | 最大5 | 時価総額 |
| CFプラス加点 | +5 | 営業CFが全期間プラス |
| 分割フラグ | +? | 株式分割の補正 |

---

## 配当スコアの詳細

### ✔ 増配判定（据え置きも増配扱い）
```
divs[i+1] >= divs[i] → 増配扱い
```

### ✔ 減配ペナルティ（−5点 × 回数）
ただし **最新年の減配は除外**。

### ✔ 特別配当補正後のデータを使用
記念配当 → 補正 → 翌年の見かけ上の減配を防止。

---

## Problem 列（問題点）
スコアリング中に検出された問題点を `/` 区切りで記録。

例：
```
配当性向70%以上/営業CFマイナス年あり/減配あり
```

---

# 🎯 3. jpx_yield_filter.py — 最終フィルタリング

## 概要
スコアリング結果（scored_output.csv）から、  
**高配当 × 財務健全 × 成長 × 規模 × CF × 減配リスク** を満たす銘柄だけを抽出。

---

## 主なロジック

### ✔ 利回り基準（規模別）
- 超大型：3.5%  
- 大型：4.0%  
- 中型：4.5%  
- 小型：5.0%  
- 超小型：6.0%

---

### ✔ 2年利回り統計（平均・最高）
- 平均利回り以上 → △  
- 平均 ×1.2 → 〇  
- 平均 ×1.35 → ◎  
- 過去最高利回り以上 → ◎  

---

### ✔ Problem が2つ以上ある銘柄は除外
```
len(Problem.split("/")) <= 1 の銘柄のみ採用
```

---

### ✔ 最新年の減配は無視
配当確定前の誤判定を防止。

---

### 出力ファイル
```
yield_filtered_output.csv
```

---

# 📄 出力ファイル一覧

| ファイル名 | 内容 |
|------------|------|
| output_financial_dividend_merged.csv | 10年配当＋財務データ |
| scored_output.csv | スコアリング結果（Problem 列含む） |
| yield_filtered_output.csv | 最終候補銘柄 |
